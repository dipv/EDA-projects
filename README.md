# EDA-projects

Data Analysis EDA Projects
This repository showcases Exploratory Data Analysis (EDA) on Superstore and Titanic datasets using Python (Pandas, Seaborn, Matplotlib).
Datasets:kaggle(Source)

Projects
Superstore EDA: Revenue trends by category (Technology ~$836K), region (West leading), and season (peaks Nov-Dec).
Titanic Survival: Survival by gender (74% females) and class (63% 1st class).

Requirements
Python 3.x
Install: pip install pandas seaborn matplotlib
